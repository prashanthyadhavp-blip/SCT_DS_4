# SCT_DS_4 – Exploratory Data Analysis (EDA)

## 📌 Task Objective
Perform Exploratory Data Analysis (EDA) on the Bank Marketing dataset to understand customer characteristics, identify patterns, and generate meaningful business insights using Python.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## 📂 Dataset
Bank Marketing Dataset (`bank.csv`)

## 📊 Analysis Performed
- Loaded the dataset
- Checked dataset information
- Identified missing values
- Checked and removed duplicate records
- Generated summary statistics
- Saved the cleaned dataset
- Created visualizations for data analysis

## 📈 Visualizations
- Job Distribution (Bar Chart)
- Deposit Distribution (Pie Chart)
- Age Distribution (Histogram)

## 💡 Key Insights
- Most customers belong to a few major job categories.
- The majority of customers did not subscribe to a term deposit.
- Customer ages are concentrated in the middle-age group.
- The cleaned dataset is ready for further predictive analysis.
